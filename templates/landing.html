What is Better Simple?

It's a system that allows you to provide an online reservation feature to your customers on your own website. It also allows you to manage those reservations online and download them to your own computer.

What will my customers see?

They will see a button that says, "Make Reservation". When they click on the button, a dialog will appear to them and ask them for the details of their reservation request. After they provide the requested information they will submit the request and receive a confirmation email.

What kind of reservations will this work for?

It currently supports appointments and trips. Where the appointment requests the customer's name, email, phone and then the appointment date and time while the trip will ask for the same things, as well as addresses for the pickup and dropoff.

Is anything customizable?

Yes, a few things are. You can choose to add special instructions and/or allow more than one attendee. The trip type reservations allows for you to specify pre-defined addresses (known as landmarks) for the user to pick from in a dropdown list.

How do I integrate with it?

After you sign up, you will be provided with some JavaScript and HTML. This code will need to be included on the page that you want the feature to appear on. The JavaScript will go in the head tag. The HTML should be inserted in a position of the file so that the page still looks aesthietcally pleasing, but also intuitive.
